# Карты и эффекты

Ниже перечислены все карты из текущего билда и то, что они делают по коду в `src/game/cards/*.ts`.

## Карты

| Карта | Эффект |
| --- | --- |
| `ЗАЖИМ` | Ремонтирует на 3. |
| `СКАН` | Даёт `×2` к ближайшей ремонтной карте. |
| `ОХЛАД.` | Снижает жар на 3. |
| `ЗАПЛАТКА` | Ремонтирует на 4. |
| `СВАРКА` | Ремонтирует на 7 и добавляет `+3` жара. |
| `ПРОДУВКА` | Сбрасывает жар до `0`. |
| `ДИАГН.` | Добирает 2 карты. |
| `ИМПУЛЬС` | Ремонтирует на `текущий жар` и добавляет `+2` жара. |
| `ЛАЗЕР` | Наносит 10 урона и добавляет `+5` жара. |
| `РАЗМЕТКА` | Даёт эффект `Следующие 2 ремонта +1`. |
| `РАДИАТОР` | В конце следующих 2 циклов снижает жар на `1`. |
| `БАЙПАС` | Снижает жар на `2` и добирает 1 карту. |
| `ДРОН` | В конце цикла наносит `+2` ремонта. |
| `КРАСНАЯ ЗОНА` | Пока жар `7+`, ремонт усиливается на `+2` на 2 цикла. |
| `РЕЛЕ` | Следующая карта повторяется. |

## Стартовая колода

- `3× ЗАЖИМ`
- `1× СКАН`
- `1× ОХЛАД.`
- `2× ЗАПЛАТКА`
- `1× СВАРКА`

## Примечания по отложенным эффектам

- `Следующие 2 ремонта +1` у `РАЗМЕТКИ` - срабатывает на 2 следующих ремонтных карты.
- `СКАН` - даёт эффект `Следующий ремонт удваивается`, который сразу применяется к ближайшей ремонтной карте.
- `РАДИАТОР` - создаёт эффект `В конце 2 циклов: -1 жара`, который срабатывает на каждом из двух следующих завершений цикла.
- `В конце цикла наносит +2 ремонта` у `ДРОНА` - срабатывает один раз в конце цикла.
- `Пока жар 7+, ремонт усиливается на +2` у `КРАСНОЙ ЗОНЫ` - действует 2 цикла и только пока текущий жар не ниже порога.
- `Следующая карта повторяется` у `РЕЛЕ` - повторяет следующую сыгранную карту один раз.
