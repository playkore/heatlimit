# Карты и эффекты

Ниже перечислены все карты из текущего билда и то, что они делают по коду в `src/game/cards/*.ts`.

## Карты

| Карта | Эффект |
| --- | --- |
| `ЗАЖИМ` | Ремонтирует на 3. |
| `СКАН` | Даёт `×2` к первой ремонтной карте следующей руки. |
| `ОХЛАД.` | Снижает жар на 3. |
| `ЗАПЛАТКА` | Ремонтирует на 4. |
| `СВАРКА` | Ремонтирует на 7 и добавляет `+3` жара. |
| `ПРОДУВКА` | Сбрасывает жар до `0`. |
| `УДАР` | Ремонтирует на 5 и добавляет `+1` жара. |
| `ДИАГН.` | Добирает 2 карты. |
| `ТЕРМОП.` | Снижает жар на 1 и даёт 2 щита цикла. |
| `ИМПУЛЬС` | Наносит `1` урон за каждый текущий жар и добавляет `+2` жара. |
| `КРИО` | Сбрасывает жар до `0` и сгорает после розыгрыша. |
| `ЛАЗЕР` | Наносит 10 урона и добавляет `+5` жара. |
| `ФОКУС` | Даёт эффект `Следующий ремонт +4`. |
| `КРИО-ПОДГОТОВКА` | Даёт эффект `Следующая горячая карта: -2 жара`. |
| `РАЗМЕТКА` | Даёт эффект `Следующие 2 ремонта +1`. |
| `ИЗОЛЕНТА` | Снимает эффект `искрение`. |
| `РЕЗОНАНС` | Даёт эффект `Следующий ремонт удваивается`. |
| `ДРОН` | В конце цикла наносит `+2` ремонта. |
| `КРАСНАЯ ЗОНА` | Пока жар `7+`, ремонт усиливается на `+2` на 2 цикла. |
| `РЕЛЕ` | Следующая карта повторяется ещё раз. |
| `КОНДЕНСАТОР` | При снижении жара даёт `+1` ремонта, действует 2 цикла. |
| `АВАРИЙНЫЙ СБРОС` | Сбрасывает жар до `0` и сгорает после розыгрыша. |
| `МАГНИТНЫЙ ЗАХВАТ` | Добирает 1 карту и даёт следующей карте эффект `-1` жара. |
| `БАЙПАС` | Следующая карта игнорирует броню. |

## Примечания по отложенным эффектам

- `Следующий ремонт +4` у `ФОКУСА` - разовый бонус к ближайшей карте с тегом `repair`.
- `Следующая горячая карта: -2 жара` у `КРИО-ПОДГОТОВКИ` - снижает жар у следующей карты с тегом `heat`.
- `Следующие 2 ремонта +1` у `РАЗМЕТКИ` - срабатывает на 2 следующих ремонтных карты.
- `Следующий ремонт удваивается` у `РЕЗОНАНСА` - разовый множитель к ближайшему ремонту.
- `СКАН` - даёт эффект `Следующая ремонтная карта следующей руки ×2`, который становится `Следующая ремонтная карта ×2` после завершения текущей руки.
- `В конце цикла наносит +2 ремонта` у `ДРОНА` - срабатывает один раз в конце цикла.
- `Пока жар 7+, ремонт усиливается на +2` у `КРАСНОЙ ЗОНЫ` - действует 2 цикла и только пока текущий жар не ниже порога.
- `Следующая карта повторяется` у `РЕЛЕ` - повторяет следующую сыгранную карту один раз.
- `При снижении жара даёт +1 ремонта` у `КОНДЕНСАТОРА` - срабатывает при любом снижении жара и действует 2 цикла.
- `Следующая карта игнорирует броню` у `БАЙПАСА` - применяется к ближайшей сыгранной карте один раз.
- `Следующая карта: -1 жара` у `МАГНИТНОГО ЗАХВАТА` - разовый эффект на ближайшую карту.
